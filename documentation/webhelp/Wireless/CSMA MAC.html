4.6 Carrier Sense Multiple Access (CSMA) MAC Protocol
4.6.1 Description
Carrier Sense Multiple Access (CSMA) is a generic carrier-sensing protocol. When a radio wishes to send data, it senses the channel. If the channel is busy, it backs off for a random time period before sensing the channel again. If the channel is free, the radio transmits the packet.
4.6.2 Command Line Configuration
To specify CSMA as the MAC protocol, include the following parameter in the scenario configuration (.config) file:
[<Qualifier>] MAC-PROTOCOL CSMA
 
The scope of this parameter declaration can be Global, Node, Subnet, or Interface. See Section 1.2.1.1 for a description of <Qualifier> for each scope.
