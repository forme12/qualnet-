4.5 Aloha MAC Protocol
4.5.1 Description
Aloha is a wireless random access MAC protocol that transmits data whenever there is data to send (no carrier sensing). If no acknowledgement is received, the source retransmits at a later time.
4.5.2 Command Line Configuration
To specify Aloha as the MAC protocol, include the following parameter in the scenario configuration (.config) file:
[<Qualifier>] MAC-PROTOCOL ALOHA
 
The scope of this parameter declaration can be Global, Node, Subnet, or Interface. See Section 1.2.1.1 for a description of <Qualifier> for each scope.
