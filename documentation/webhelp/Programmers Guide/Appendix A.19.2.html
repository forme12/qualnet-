A.1 Introduction
The major differences between the 32-bit and the 64-bit QualNet development environments arise from the fact that the 32-bit QualNet implementation is based on the ILP32 data model, whereas the 64-bit QualNet implementation is based on the LP64 data model (for Linux systems) or the P64 data model (for Windows systems). In the ILP32 data model, long and pointer types are 32 bits long, where as in the LP64 and P64 data models, long and pointer types are 64 bits long. The other fundamental data types have the same length in the 32-bit and 64-bit data models.
