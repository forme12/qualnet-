C.1 Local Execution
To set the run mode for local execution in QualNet GUI, do the following:
1.
Open the Run Settings window (see Figure C-4) by pressing the Run Settings button (which is to the left of the Run Simulation button).
FIGURE C-1.  Run Settings Button
FIGURE C-2.  Run Settings for Local Execution
2.
Select the Local Execution radio button.
3.
Set Number of Processors to the number of processors on which to run QualNet.
4.
Check the Distributed Run Mode option to run QualNet on a distributed platform, and in the Host File field that is displayed, enter the name of the host file, which lists the computers to be used. Refer to QualNet Distributed Reference Guide for details of the host file.
 
Note: 
The Distributed Run Mode option is available only if QUALNET_HOME/bin contains a distributed-mode executable (qualnet.mpi). Refer to QualNet Distributed Reference Guide for details of compiling QualNet on a distributed platform.
5.
Press OK to save the changes.
These run settings will be used for all subsequent simulations.
