Advanced Run Settings
Using the Run Settings dialog, the QualNet GUI can be configured to run a simulation in the following modes:
•
locally, using a single processor
•
locally, in parallel mode on a shared-memory multi-processor system
•
locally, in parallel mode on a distributed platform (refer to QualNet Distributed Reference Guide for the requirements for running QualNet on a distributed platform)
•
remotely, with the simulator running on a server in any of the above modes
