Modeling Networks
This chapter describes how to create networks models in QualNet (in both command line and GUI) and how to use QualNet for emulation.
Section 4.1 describes conventions used in this chapter.
Section 4.2 gives a detailed description of configuring network scenarios. It describes how to configure high-level parameters in command line and in Architect. References to model libraries are provided to configure model-dependent parameters.
Section 4.3 describes the parameters for configuring QualNet for a multi-core/multi-processor environment.
Section 4.4 describes the parameters for achieving trade-off between simulation speed and accuracy of results.
Section 4.5 gives an overview of advanced network modeling features. These features are described in detail in the model libraries.
