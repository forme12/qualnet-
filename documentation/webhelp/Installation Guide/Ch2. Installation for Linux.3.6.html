2.5 Compiling QualNet
The QualNet installer places a pre-compiled executable file in QUALNET_HOME\bin, where QUALNET_HOME is the directory where QualNet is installed. This pre-compiled executable can be used directly and you do not need to compile QualNet. Refer to QualNet User’s Guide for details.
However, if you modify the code or activate certain libraries, you will need to recompile QualNet. Refer to QualNet Programmer’s Guide for details.
