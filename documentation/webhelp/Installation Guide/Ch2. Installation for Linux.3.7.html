2.6 Uninstalling QualNet
To uninstall QualNet, do the following:
1.
Open a command window.
2.
Change directories to any location other than the QualNet installation directory or any of its subdirectories.
3.
Type the following command to uninstall QualNet from the local machine:
 
$QUALNET_HOME/uninstall
 
Type the following command to uninstall QualNet from a remote machine:
 
$QUALNET_HOME/uninstall --mode text
