3.2 Phone Call Traffic Generator
3.2.1 Description
The Phone Call traffic generator is an application that simulates phone calls between two end users. The initiator places a call to the receiver and talks for a period of time during which the receiver is silent. After the initiator has finished talking, the receiver talks for a period of time during which the initiator is silent. The call continues with the initiator and receiver talking during alternate talk periods.
 
Note: 
In UMTS networks, a Phone Call session can be configured only between two UE nodes.
3.2.2 Supplemental Information
None.
3.2.3 Command Line Configuration
To specify the Phone Call application, include the following statement in the application configuration (.app) file:
PHONE-CALL <initiator> <receiver> <avg-talk-time> <start-time>
<duration>
 
Note: 
All parameters should be entered on the same line.
The Phone Call parameters are described in Table 3-4. See Section 1.2.1.3 for a description of the format used for the parameter table.
 
TABLE 3-4.  Phone Call Parameters 
Parameter
Value
Description
<initiator>
Required
Integer
Node ID or IP address of the initiator of the call.
<receiver>
Required
Integer
Node ID or IP address of the receiver of the call.
<avg-talk-time>
Required
Time
Average length of a talk period (the time during which one party talks and the other party listens).
The length of the talk period is taken from an exponential distribution with <avg-talk-time> as the mean.
<start-time>
Required
Time
Simulation time when the call starts.
<duration>
Required
Time
Total length of the call.
A typical value for this parameter is in the range 180S to 210S.
Examples of Command Line Configuration
In the following PHONE-CALL configuration, node 5 places a call to node 8 10 seconds after the start of simulation. The average length of a talk period is 15 seconds. The phone call ends after 100 seconds.
 
PHONE-CALL 5 8 10S 15S 100S
3.2.4 GUI Configuration
To configure a Phone Call session between two nodes, perform the following steps:
1.
Click the UMTS button in the Applications tab.
2.
On the canvas, click on the initiator node, drag the mouse to the receiver node, and release.
3.
Open the UMTS Call Properties Editor by doing one of the following:
a.
Right-click in the application link on the canvas and select Properties from the menu.
b.
In the Applications tab of Table View either double-click on the application row or right-click on the application row and select Properties from the menu.
4.
Set the parameters listed in Table 3-5.
