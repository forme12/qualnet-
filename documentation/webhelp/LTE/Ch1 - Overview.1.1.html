Overview of Model Library
1.1 Overview
The Long Term Evolution (LTE) Model Library provides high fidelity simulation of 4G cellular networks based on the 3GPP Release 9 standards. It provides system-level scalability, fast execution speeds and detailed MAC and PHY modeling. It includes models of nodes called eNodeB (Base Station) and UE (Mobile Station).
