3.5 Creating an Application Session
The scenario has one Constant Bit Rate (CBR) application session between nodes 1 and 2. To specify a CBR session, do the following:
1.
Click on the CBR button in the Applications toolbar (see Figure 3‑4).
2.
Click on node 1, drag the mouse to node 2, and release. A green arrow labelled CBR from node 1 to node 2 will be drawn on the canvas.
