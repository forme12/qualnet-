3.7 Saving and Running Scenario
Save and run the scenario by doing the following:
1.
Save the scenario by clicking the Save button in the toolbar (see Figure 3‑4).
2.
Initialize the scenario by clicking the Run SImulation button. Start the simulation by clicking the Play button. See Chapter 2 for details of running the scenario and analyzing the results.
