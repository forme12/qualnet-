Error code:    -14
               Cannot find SERVER hostname in network database. The lookup for
               the hostname on the SERVER line in the license file failed. This
Error message: often happens when NIS or DNS or the hosts file is incorrect.
               Workaround: Use IP-Address (e.g., 123.456.789.123) instead of
               hostname
