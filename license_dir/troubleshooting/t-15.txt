Error code:    -15
               Cannot connect to license server. The server (lmgrd) has not
Error message: been started yet, or the wrong port@host or license file is
               being used, or the port or hostname in the license file has been
               changed.
               Ensure that license server is started. Make sure you can ping to
               the license server from the client machine.
Solution:      If the issue persists, switch off firewall / sercurity settings
               that may cause unreachability.
               Related FAQ: http://support.scalable-networks.com/categories/20014366-license-support
